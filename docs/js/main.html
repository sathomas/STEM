    Main JavaScript file for STEM Incubator site. Code in this file mostly just sets things up for the app and then launches it. This file also includes configuration information.

    After setup completes, control transfers to the the single instance of an app router.

    Everything in the application is namespaced under the global variable Stem. Invidual components (e.g. models, views, collections, etc.) add themselves to this namespace as part of their definition.

    window.Stem = {
    Collections: {},
    Models: {},
    Routers: {},
    Views: {},
